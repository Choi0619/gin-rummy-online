🃏 진 러미 플레이 방법

🎯 목표

손에 든 카드를 멜드(meld)로 정리해서 데드우드(meld에 속하지 않은 카드 점수 합)를 최대한 줄이는 게임이에요. A=1점, 2~10은 숫자 그대로, J·Q·K는 10점입니다.

🧩 멜드 종류

세트(Set) — 같은 숫자 3~4장, 모양은 모두 달라야 함

7♠7♥7♣

런(Run) — 같은 모양 3장 이상, 숫자가 연속이어야 함

4♠5♠6♠

🔄 턴 진행

  • 덱 🂠 또는 버린 패 ♻️ 중 한 곳에서 카드 1장을 가져옵니다
  • 손에서 카드 1장을 선택해 버립니다 (멜드는 초록 테두리로 자동 표시돼요)
  • 조건이 맞으면 버리는 대신 녹/진/빅진을 선언할 수 있습니다

🏆 라운드를 끝내는 방법

🤜 녹 (Knock) 데드우드 10점 이하일 때 선언. 상대가 내 멜드에 카드를 얹는 레이오프 후 점수 비교

✨ 진 (Gin) 데드우드 0점(10장 전부 멜드)일 때 선언. 레이오프 없이 25점 + 상대 데드우드 획득

💎 빅진 (Big Gin) 카드를 뽑은 11장 전부가 멜드일 때 즉시 선언 (버릴 필요 없음!). 31점 + 상대 데드우드 획득

🔄 언더컷 녹 선언자보다 상대 데드우드가 같거나 낮으면 상대가 역전! +25점 보너스

🏳️ 항복 상대가 수락하면 그 라운드는 상대 승리(+25점)로 즉시 종료

🧮 점수 요약

상황점수
진 성공25 + 상대 데드우드
빅진 성공31 + 상대 데드우드
녹 성공(상대 데드우드 − 내 데드우드)
언더컷 당함(내 데드우드 − 상대 데드우드) + 25
항복 수락됨고정 25점

👫 같이 할 때 편의 기능

  • ⏸ 자리비움 — 잠깐 자리 비울 때 켜면 AI가 대신 플레이 (복귀하면 자동 해제)
  • 🛑 게임 종료 요청 — 상대 동의 시 현재 점수로 매치 종료
  • 🚪 나가기 — 상대에게 퇴장 알림 후 즉시 방 종료
